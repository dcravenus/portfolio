Projects I've Worked On

TV Showtimes

I wanted an easier way to view when my favorite TV shows air new episodes so I built a simple TV Showtimes app. The app uses the TVmaze API to load basic metadata about different TV shows. The app stores all its data on the client to reduce API calls and increase page load speed.

Hummedia

The Hummedia project allows instructors to host videos with custom annotations. Users can annotate videos with interactive text, images and video; use translation tools to learn new languages; and select scenes to skip among many other features.

Automatic Pronunciation Evaluation (APE)

The Automatic Pronunciation Evaluation (APE) project allows foreign-language learners to receive visual feedback to help improve their pronunciation. The VowelWorm is a real-time version of the project that uses JavaScript to give immediate results in the browser.
